class UserBook(models.Model):
    user = models.ForeignKey(User, on_delete=models.CASCADE)
    book = models.ForeignKey(Book, on_delete=models.CASCADE)

UserBook is the child, while user and book are the parents
Foreignkey() creates a ManytoOne relatiosnhip i.e a single user can have many UserBooks
